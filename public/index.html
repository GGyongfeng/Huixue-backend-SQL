汇学教育 API 服务

欢迎访问 api.guyongfeng.cn，这是汇学教育系统的 API 服务器。

API 服务概览

管理端接口

POST /api/manager/users/login - 管理员登录

GET /api/manager/menu/list - 获取菜单列表

GET /api/manager/tutors/list - 获取家教订单列表

POST /api/manager/tutors/create - 创建家教订单

PUT /api/manager/tutors/update/:id - 更新家教订单

POST /api/manager/tutors/delete - 删除家教订单

教师端接口

GET /api/teacher/tutorslist - 获取可接订单列表

GET /api/teacher/profile - 获取教师信息

GET /api/teacher/tutornotice - 获取系统通知

技术特点

  • 基于 Express.js 框架开发
  • 支持多城市数据库独立部署
  • JWT 认证保护
  • 完整的错误处理机制
  • 支持跨域请求
